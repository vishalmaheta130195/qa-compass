Standard Data Generator

v1.4

Fills the gap between "I need a fake VAT number for Belgium" and actually finding one. Click the extension on any page, scan its fields, and generate correctly formatted test data — tax/VAT numbers, national IDs, IBAN, postal codes, passport numbers, bank accounts, mobile numbers, email, names, age, and date of birth — across 36 countries.

For VAT numbers, national IDs, and IBAN, it doesn't just format digits — it implements the real public check-digit algorithms (the same ones used by Belgium, the Netherlands, France, Spain, all EU countries, Australia, Brazil, and India's Aadhaar via the Verhoeff algorithm), so generated values pass the same validation a real one would. Where no public checksum exists, values are clearly labeled format-only.

Broken Link Checker

v1.0

Checks every link on the page you're currently viewing and reports which ones are broken. Click the extension, hit Scan, and it collects every link on the page, checks each one's HTTP status, and shows a report of what's working and what's not — no more manually clicking through every link on a page during a QA pass.
